# Installation Guide - SecureCloud Pay

## Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- MongoDB or PostgreSQL (optional for local development)
- Git

## Project Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Kirantechsavvy/SecureCloud-pay.git
cd SecureCloud-pay
```

### 2. Install Dependencies

#### Backend Setup
```bash
cd backend
npm install
```

#### Frontend Setup
```bash
cd ../frontend
# Frontend doesn't require npm install for vanilla JS
```

### 3. Environment Configuration

Create a `.env` file in the backend directory with the required configuration.

### 4. Run the Application

#### Backend
```bash
cd backend
npm run dev
```

#### Frontend
```bash
Open frontend/index.html in your browser
```
